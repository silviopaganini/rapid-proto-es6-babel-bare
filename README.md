# Rapid prototyping
## ES6 + Babel Bare bones

### Presets
- ES2015
- Class properties 

### Usage
```bash
npm install
npm start
```

### Build
Creates a build folder with `index.html`, `style.css` and `bundle.js` from `./src`

```bash
npm run build
npm run build:run #builds and start local server to test the build
```

### Test

ESLint for JS
```bash 
npm test
```

### Lint & fix
```bash 
npm run lint:fix
```
